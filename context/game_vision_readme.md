## 📋 Пошаговый план интеграции

### Шаг 1. Создание проекта Supabase
1. Перейди на [Supabase](https://supabase.com) и зарегистрируйся (через GitHub).
2. Нажми **New Project**, введи название (`game-vision`) и придумай надежный пароль для базы данных.
3. Выбери ближайший регион (например, `Frankfurt`).
4. Дождись завершения создания (~2 минуты).
5. В левом меню открой **Project Settings → API** и скопируй два ключа:
   - `Project URL` (например, `https://abcdefgh.supabase.co`)
   - `anon public` (публичный ключ)

### Шаг 2. Настройка базы данных
1. В панели Supabase открой раздел **SQL Editor**.
2. Скопируй и выполни скрипт из нашего файла `code_snippets/supabase_schema.md`.

### Шаг 3. Настройка конфигурации фронтенда
1. Открой файл `js/config.js` в проекте.
2. Вставь свои ключи от Supabase (`SUPABASE_URL` и `SUPABASE_ANON_KEY`).

### Шаг 4. Настройка Auth Redirect (Важно для входа!)
1. В Supabase открой **Authentication → URL Configuration**.
2. В поле **Site URL** пропиши адрес твоего сайта (например, `http://localhost:8000` для локальной разработки или ссылку с Vercel/Netlify после деплоя).
