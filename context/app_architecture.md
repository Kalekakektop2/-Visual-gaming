# Архитектура приложения: Game-Vision Analyzer (Десктопная часть)

**Суть проекта:** Десктопное приложение для геймеров, которое производит анализ игрового процесса (захват экрана/видео, обработка кадров, взаимодействие с ИИ-моделями для подсказок или анализа статистики).

## 🧩 Компоненты системы:
1. **Веб-интерфейс (уже готов):** Лендинг, авторизация (Supabase Auth), управление подписками (`subscriptions` в Supabase).
2. **Десктопная программа (нужно разработать):**
   - Написана на [укажи язык, например: Python / Electron / C#].
   - **Авторизация:** Должна запрашивать логин/пароль или токен из веб-версии, проверять через Supabase, активна ли подписка (`status == 'active'` и не истек ли `expires_at`). Если подписка просрочена — блокировать функционал анализа.
   - **Основной функционал:** Захват экрана игры, отправка кадров/данных на анализ и вывод результатов игроку.
